## 可灵AI图片生成字段插件

### 功能描述
基于可灵AI大模型，支持高质量图片生成，包括文生图、图生图、多图融合等功能。

### 使用介绍
- **文生图**: 输入提示词生成图片
- **图生图**: 上传参考图片，基于提示词进行图片编辑
- **多图融合**: 支持最多9张参考图片融合生成新图片
- **组图模式**: 支持生成2-9张组图

### 支持模型
- kling-image-o1: 可灵图像模型
- kling-v3-omni: 可灵全能模型

### 参数说明
- **提示词**: 不能超过2500个字符，支持通过 <<<>>> 格式指定图片位置
- **参考图片**: 支持 .jpg/.jpeg/.png 格式
- **画面比例**: 支持自动、1:1、16:9等多种比例
- **清晰度**: 1k/2k/4k 三档可选
- **输出模式**: 单图或组图
